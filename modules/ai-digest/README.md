# ai-digest

## Context

Most solo builders want a daily/weekly AI update but cannot spend hours filtering noisy content.

## Goal

Build a simple AI digest pipeline that:

- collects selected sources
- summarizes key updates
- outputs a readable markdown digest

Target: first useful output in <= 30 minutes.

## Scope (v0 skeleton)

- Input: manually curated links/text snippets
- Process: summarize and structure by theme
- Output: `digest.md`

## Planned Deliverable

- `input/sample_sources.md`
- `output/digest.md` (generated)
- step-by-step run guide

## Success Criteria

- User can generate one digest end-to-end
- Digest has clear sections (news, tools, practical takeaways)
- Output is reusable for newsletter/social post drafting
