# Modules

Prax modules are practical learning units designed for fast execution and real output.

## Current Phase 1 Modules

- `ai-digest`
- `content-repurpose`
- `workflow-starter`

## Module Contract

Each module should follow the same structure:

1. Context
2. Goal
3. Steps
4. Run
5. Extend

And include:

- expected output example
- common errors and fixes
- one extension task
